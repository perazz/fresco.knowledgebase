Flame Speeds

Flame speed input#

The user can currently only choose between a selection of pre-tested laminar flame speed correlations. A laminar flame speed is specified in itapeFlame with the following syntax:

speed [model] [choice] [(attachedSpecies)]

where:

- *model* defines the laminar flame speed model, which can be:
- `DISTASO` to use the predefined correlations in [Distaso and Reitz](#1)
- `METGHALCHI` to use the predefined correlations in [Metghalchi and Keck](#2)
- `CONSTANT` to specify a constant flame speed
- `HYDROGEN` to enable the Hydrogen flame speed model.

Matlab flame speed initialization#

In the Matlab structure, flame speed initialization is still based on a model/choice flag, done via a call to the routine fresco_flame_speed_get. For example:

flame.speed = fresco_flame_speed_get('METGHALCHI','PROPANE');
